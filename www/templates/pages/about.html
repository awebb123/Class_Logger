What happens behind the scenes
Example Button Example Checkbox
Disabled Button Disabled Checkbox

The buttons you see are built using Angular Material. In our code, we use md-button to create a button. To make a button disabled, we use ng-disabled and set the value to true. The checkboxes are built similarly, however we use md-checkbox to create them. To make a checkbox disabled, we use ng-disabled and set the value to true, like we did for the button.
